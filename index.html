Featured Work

Slip of the Hand

A vanilla JS project that accepts a word and using an algorithm considers all strings that could have been generated from one or both misplaced hands. It then makes a call to a dictionary API and returns all positive hits. Hands/keys are also programed to mimic user entry. Slip Of The Hand!!

Renoun

Renoun is a pixel perfect clone of Reverb.com, an online marketplace for instruments. It features user authentication as well as other full CRUD features including user cart, seller reviews, and instrument favoriting. The backend uses Ruby on Rails and PostgreSQL while the front end relies upon React and Redux

genie

genie is an AI powered app that generates flashcards from pdf submissions or user-specified terms/definitions. This was a group project for which I led the backend team, setting up a MongoDB/Express.js backend framework to host the flashcards, decks, and ability to add to a favorite list of decks

Hi, I'm CharLee

A seasoned educator looking to disrupt the online learning industry

I am a software engineer with 20 years experience in the field of education with a focus on economics and business.
